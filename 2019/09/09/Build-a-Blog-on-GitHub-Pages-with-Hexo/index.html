Build a Blog on GitHub Pages with Hexo

Posted on 2019-09-09

Environment Details

  • Desktop OS: macOS High Sierra 10.13.6
  • Node.js: 10.16.3
  • Node.js Package Manager (NPM): 6.9.0

Prepare Hexo Environment

  • Download Node.js and install manually.

  • Install Hexo with npm

  • Initialize your blog

  • Install npm for your blog

  • After initialization, you can see your folder directories as follows:

  • Generate files for publish

  • Start hexo server on localhost:4000

Publish to GitHub Pages

  • Create a public GitHub repository

  • Install hexo-deployer-git

  • Set deployment information in _config.yml

  • Publish to GitHub repository

  • Enable GitHub Pages for repository (Check Here for more details)

  • Your Hexo blog is now available.
